## Project Overview

BudgetBuddy is a personal budgeting and expense management application developed for Android devices. The purpose of the application is to assist users in managing their finances by tracking income and expenses, creating spending categories, setting budget goals, and viewing financial reports.

The application was designed to provide users with a simple and user-friendly interface while offering useful financial insights through charts, statistics, and progress tracking.

---

## Main Features

### User Authentication

* Secure login functionality
* Username and password validation

### Category Management

* Create and manage expense categories
* Organise transactions effectively

### Expense Tracking

* Record income and expenses
* Add descriptions to transactions
* Select transaction dates
* Assign transactions to categories

### Photo Attachments

* Attach photographs to expense entries
* View saved images from transaction records

### Budget Goals

* Set minimum monthly spending goals
* Set maximum monthly spending goals
* Monitor progress against goals

### Financial Reports

* View spending summaries
* Analyse expenses by category
* Filter reports by selected date ranges

### Graphical Analysis

* Spending per category graphs
* Budget goal comparison graphs
* Monthly spending visualisations

### Gamification Features

* Achievement badges
* User streak tracking
* Reward system for consistent usage

### Additional Features

* Dashboard overview
* Monthly savings calculation
* Budget progress monitoring
* Category spending breakdown
